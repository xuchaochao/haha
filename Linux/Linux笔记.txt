sudo apt-get update 更新当前系统的软件包。
sudo gedit 某文件名表示用某文本编辑器打开和查看某文件，gedit是GNOME桌面环境下兼容UTF-8的文本编辑器

清屏：clear

dpkg:
dpkg -L package 查看软件安装到什么地方
dpkg -l package 显示包的版本
dpkg -P package 移除软件并且不保留配置

自动补全命令用tab按钮，*表示任意长度的字符串，？表示一个字符。
查看所处目录路径，pwd。
ls命令下，目录为蓝色，普通文件为黑色（或者白色），可执行文件为草绿色，淡蓝色表示这个文件是一个链接文件（相当于快捷方式）。ls -F能显示出目录，ls -a能显示出该目录下所有的隐藏文件,ls -l能查看文件的各种属性。
cat -n 某文件,能以显示行数的方式展示一个文件内容。
more命令能部分展示文件内容，然后按enter键展示更多；当然也可以使用less命令更加的好用。
head，tail -行数 能查看文件的开始或者结束的那几行。
grep 关键词 文件名或者grep ‘关键词（对于有空格的情况用引号）’ 文件名
find 文件名就能找到当前目录下的文件（一般用*？来搭配使用）.

locate *.desktop表示搜索某个文件所在的目录，locate用的是检索文件名数据库来确定文件的位置，如果希望获得该数据库的最新版本那么可以在root权限下用updatedb刷新一下，然后再locate。

创建目录用mkdir 目录名称，创建一个空文件touch 文件名;

移动文件或目录用mv 待移动的目录或者文件名 目标目录；当目前目录中可能存在相同名称的文件时可以使用mv -i 文件名 目标目录，这样就会有出来提示是否要覆盖；重命名用mv 当前文件 目标文件；

复制文件和目录用cp，cp 当前文件 目标目录；同样可以用cp -i处理目标目录可能会有同名称文件的情况，

rmdir可以用来删除目录，但是目录中有文件时会删除失败，可以使用rm -r来递归删除该目录下的所有文件包括目录。

文件权限r为4，x为1，w为2，rwx=4+2+1=7，chmod 777 某文件表示该文件可以被所有用户执行和读写。
linux中的文件类型有普通文件-，目录d，字符设备文件c，块设备文件b，本地域套接口s，有名管道p，符号链
接l。

建立软链接：ln -s 原文件 链接文件，从此访问链接文件就相当于访问原文件。
建立硬链接，ln 原文件 链接文件，对任意文件进行修改都会在另外一个文件中体现。

输出重定向：ls -l > n，将打印的内容重定向到一个文件中（若该文件不存在那么将建立此文件），若该文件已经存在那么会删除原文件中的所有内容，然后用新内容来代替。

输入重定向：stdin标准输入（stand input的意思），cat < 某文件，没啥用这个。

管道 | ，实例 ls -l | grep 待查找的字符，意味将ls -l输出的东西输出给grep命令，代表了多个命令可以组合使用。
